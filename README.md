## 🚀 **INITIALIZATION PROTOCOL**

### 1️⃣ **DEPLOY NEURAL HUB (BACKEND)**
```bash
cd backend
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
./run_server.sh
```

### 2️⃣ **ACTIVATE USER INTERFACE (FRONTEND)**
```bash
cd frontend
npm install --force
npm start
```
